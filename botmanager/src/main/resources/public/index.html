ACSA Chatbot

Agente conversacional para la solicitud de cita en Atención Primaria

 

Descripción

El bot permite solicitar citas para ser atendido de forma presencial o telefónica. Para ello, el bot ofrecerá al usuario varias fechas entre las cuales elegir y, una vez seleccionada la fecha, propondrá distintas franjas horarias. Las citas podrán ser modificadas o anuladas con posterioridad. Para poder realizar cualquier gestión, es preciso que el usuario se identifique mediante su DNI o NUHSA y que acepte las condiciones de uso (que pueden ser revocadas). Además, el bot ofrece información sobre el centro de salud del usuario y, en el caso de que la tuviese, de su próxima cita con su médico de atención primaria.
